# Phone-Directory-Using-BinarySearchTree
a c++ based program that uses the application of binary search tree to search for contact in a phone directory in O(log n ) time

## Features

Add new contacts with name and phone number

Search contact by name

Delete existing contacts

Display all contacts alphabetically (in-order traversal of BST)

Menu-driven console interface

## Concepts Used

Binary Search Tree (insertion, deletion, search)

Recursion and tree traversal (in-order)

String handling and user input in C++
